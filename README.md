# EncryptionPractice
## What is this?
This project was made as part of weekly tasks for Dragons SKN's CTF division.<br>
Besides simply fulfilling this one task, I decided to take this opportunity to try something new.<br>
That's why I opted to go with Rust and Qt, technologies I've been itching to learn for a long time.<br>
Since it's my first time using them and because I had to rush this project a bit, expect smelly code in some places.
## How to build
1. Make sure you have Rust/cargo and Qt 6 installed.
2. Run `cargo run`.

(Really complicated, I know)

**Warning, this program hasn't been tested on Windows!**
